# Recovery: Scan & Save

Static landing page for the Recovery: Scan & Save app.

## Files

- `index.html` - homepage
- `styles.css` - page styling
- `assets/` - image assets
- `app-ads.txt` - app ads declaration
- `test-homepage.ps1` - local content and asset checks

## App Ads

The app ads file should be served from the site root:

```text
/app-ads.txt
```

Expected content:

```text
google.com, pub-3106003114382927, DIRECT, f08c47fec0942fa0
```
